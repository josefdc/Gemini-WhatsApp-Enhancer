# ⌨️ Atajos de WhatsApp Web (Para Referencia)

## Atajos Nativos de WhatsApp que NO debemos usar:

| Atajo | Función en WhatsApp |
|-------|---------------------|
| `Ctrl+N` | Nueva conversación |
| `Ctrl+Shift+]` | Chat siguiente |
| `Ctrl+Shift+[` | Chat anterior |
| `Ctrl+E` | Archivar chat |
| `Ctrl+Shift+M` | Silenciar chat |
| `Ctrl+Backspace` | Borrar chat |
| `Ctrl+Shift+U` | Marcar como no leído |
| `Ctrl+Shift+N` | Crear nuevo grupo |
| `Ctrl+P` | Abrir perfil |
| `Ctrl+Shift+F` | **Buscar en mensajes** ❌ (Conflicto anterior) |
| `Ctrl+F` | Buscar o filtrar |
| `Ctrl+Alt+Shift+Down` | Ir a mensaje no leído |

## ✅ Nuestro Atajo (Sin Conflictos):

| Atajo | Función | Estado |
|-------|---------|--------|
| **`Ctrl+Shift+X`** | Fix grammar (Gemini) | ✅ **LIBRE** - No usado por WhatsApp |

### Por qué `Ctrl+Shift+X`:
- ✅ **X** = "fiX" (fácil de recordar)
- ✅ No está en la lista de atajos de WhatsApp
- ✅ No interfiere con Chrome
- ✅ Fácil de presionar con una mano
- ✅ Mac: `Command+Shift+X`
